Bienvenido a PharmaBusiness

Seguí estos pasos para configurar la aplicación — es completamente gratuito

1

Crear una base en Airtable

airtable.com

① Creá una cuenta gratuita en airtable.com

② Creá una base nueva con una tabla llamada exactamente Facturas

③ Agregá estas columnas (los nombres deben ser exactos):

LaboratorioTexto NumFacturaTexto FechaFecha ImporteNúmero FechaVencimientoFecha TipoOpción única NotasTexto largo PagadaCasilla

④ En airtable.com/create/tokens creá un token con permisos data.records:read y data.records:write

⑤ El Base ID está en la URL de tu base: airtable.com/appXXXXXXXX/...

2

Obtener API Key de Gemini

aistudio.google.com

① Ingresá a aistudio.google.com con tu cuenta Google

② Clic en "Get API key" → "Create API key"

③ Copiá la clave — empieza con AIza...

✓ Gratuita — no requiere tarjeta de crédito. Incluye 1.500 escaneos/día.

3

Pegá las claves en la configuración

Hacé clic en el botón de abajo para abrir el panel
